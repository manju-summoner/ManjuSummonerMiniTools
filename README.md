# ツール置き場
## CopyPngToClipboardWithAlpha
PNG画像の透過情報を維持したまま画像をクリップボードにコピーするためのツール。  
画像の貼り付け側で特別な対応が必要。
- Paint.NET
- FireAlpaka
等で利用可能。

## PastePngFromClipboardWithAlpha
PNG画像の透過情報を維持したまま画像をクリップボードから読み込むため処理を実装したサンプルプログラム。
画像のコピー側で特別な対応が必要。

## LaunchSnippingTool
Windows11のSnippingToolで指定した画像を開くためのツール。
